Training school programme

The training school will last 5 days and consists of three distinct blocks (described in detail below) in which the participants will learn how to code in the R programming language, query data from the ARIADNE Knowledge Base, and analyze it using various spatial analysis methods, especially point pattern analysis.

1. R Programming & general topics

R is a programming language and an environment for statistical computing and graphics. All the workshops of the school will use R and this practical introduction will give you enough experience to be able to write and understand code on yourself, know where and how to look for help, read and write data to/from R, and create and export basic plots.

1.1 Introduction to R and workflows

An introductory course to R. Anyone with minimal or no prior experience with R will be able to understand what is happening in the code and write or adapt existing code for their use case.

1.2 Data analysis and visualization in R

You will learn basics of data manipulation with dplyr package and data visualization with ggplot2 package as well as some basics of base R. This will give you a powerfull set of tools to analyze and visualize data.

1.3 Introduction to spatial data in R

You will learn how spatial data is represented in R, especially vector data as simple features in the sf package and raster data in the terra package. We will also have a look at various operations with geographic data, including reprojecting, geometry operations etc. Integral part of the workshop will be making maps in R.

1.4 Reproducibility and good practice in data and code management

2. ARIADNE Data

ARIADNE infrastructure aggregates and integrates large amounts of archaeology data worldwide. Second day of the summer school will be focused on how the data is represented in the ARIADNE Knowledge Base (ARIADNE KB) and how to access it.

2.1 ARIADNE & AO-Cat Ontology

We will start by exploring ARIADNE Portal, a catalogue interface to the ARIADNE KB. After building up notion of how data look in the ARIADNE Portal, we will delve deeper into how data is represented in the ARIADNE KB, from general notion of linked-open data to semantic data in RDF. Finally, we will explore the AO-Cat Ontology, which will give us a basis for querying the knowledge base.

2.2 Introduction to SPARQL

SPARQL is a query language for semantic data in RDF. You will learn very basics of SPARQL, how do the queries look, what parts are there etc. Practical examples of queries performed on the ARIADNE SPARQL endpoint will help us to understand what is possible with the query language.

2.3 Querying ARIADNE SPARQL endpoint

In the workshop, you will combine the knowledge of AO-Cat ontology and SPARQL to query ARIADNE KB. At the end of the day you will be able to create and adapt various queries to obtain data from the ARIADNE KB for further analysis and re-use.
